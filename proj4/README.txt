simple photomanager using PHP/MySQL back-end

setup on CSELabs webserver
1) upload all files in chmod 701 directory
2) chmod 701 index.php; chmod 701 dbconfig.php; chmod 604 favicon.ico; chmod 604 script.js; chmod 604 style.css
3) access with web-browser

NOTE: login for photo manager is prefilled with correct values [username, password]

WARNING: The cookie auth used in this bundle is very weak.
